    • Chapter 3

  • Array code Example:
    const cars = [];
    cars[0] = 'Ford';
    cars[1] = 'Toyota';
    cars[2] = 'Honda';
    delete cars[1]; // This will replace Toyota with "undefined".
    cars.length; // This will display a number corresponding to the number of items in an array.
    cars.pop(); // This will remove the last variable in the array.
    cars.shift(); // This will remove the first variable in the array.
    cars.push('Volkswagon'); // This will add a variable to the end of an array.
    cars.unshift('Volkswagon'); // This will add the variable to the front of the array.
    cars.concat(['Jeep', 'Subaru']); // This will combine the defined array with cars array.
    cars.join(' & '); // This will turn the array into a string.
  • There are other examples such as reverse, sort, indexOf, includes and many others
  • Coordinates can be made with 2d arrays.
  • Sets are groups of individual characters or numbers.
    For example: const numbers = new Set([1,2,3]);
  • Sets can be converted to arrays by using the Array.from() method
  • Maps are a different structure similar to objects. Objects are limited to using strings
    as key values, whereas maps can use any data type.
