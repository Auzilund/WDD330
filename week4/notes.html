Object Oriented Programming

  • Object-oriented programming (OOP) is a way of programming that uses
    objects that encapsulate their own properties and methods.
  • All instances of a class or constructor function inherit all
    the properties and methods of its prototype.
  • Inside a constructor function or class declaration, the
    keyword "this" refers to the object returned by the function.
  • Methods can be borrowed from other objects.
